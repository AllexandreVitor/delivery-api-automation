# 🚀 Delivery API Automation - CI/CD com GitHub Actions

## 📌 Visão Geral

Este projeto tem como objetivo a implementação de uma esteira de **Integração Contínua (CI)** e práticas de **DevSecOps** utilizando **GitHub Actions**.

A aplicação base utilizada é a **Delivery API (Node.js)**, onde foi configurado um pipeline automatizado para garantir qualidade, segurança e confiabilidade do código antes de qualquer integração na branch principal (`main`).

---

## 🎯 Objetivo da Atividade

Consolidar conceitos de:

- Integração Contínua (CI)
- DevSecOps
- Automação de pipelines
- Qualidade de código
- Segurança em aplicações Node.js

---

## ⚙️ Tecnologias Utilizadas

- Node.js v24
- GitHub Actions
- npm
- njsscan (SAST - Static Application Security Testing)
- Ubuntu Latest (runner)
